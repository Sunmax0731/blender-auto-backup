# v0.1.0 - Blender Auto Backup MVP

Release date: 2026-05-14

## Summary

Blender Auto Backup is a Blender 4.2+ add-on that creates ZIP backups of a selected working folder on demand or on a timer. This first public release includes the MVP backup workflow plus the completed local backlog for background execution, include / exclude filters, global default backup location, runtime gate diagnostics, Steam Blender detection, and selectable destination layout.

## Included Features

- Manual ZIP backup from Scene Properties > Auto Backup.
- Timer-based automatic backup.
- Optional background worker execution for large folders.
- Include and exclude glob rules.
- Global default backup folder in add-on preferences.
- Direct destination layout and project subfolder destination layout.
- Backup destination exclusion when the backup folder is inside the source folder.
- `.partial` archive creation before final ZIP replacement.
- Retention cleanup based on `Max Backups`.

## Install

1. Download `blender-auto-backup-0.1.0.zip` from the GitHub release assets.
2. In Blender, open `Edit > Preferences > Add-ons`.
3. Use `Install from Disk...` and select the ZIP.
4. Enable `Blender Auto Backup`.
5. Configure `Source Folder`, `Backup Folder`, and `Destination Layout` from Scene Properties > Auto Backup.

## Known Notes

- The add-on is file-system only and does not expose a web URL.
- Blender 4.2 or later is required.
- Blender 5.1.1 is the user-side runtime validation target for this release.
